About

What is TOPLAP HK?

TOPLAP (Temporary Organisation for the Proliferation of Live Algorithm Programming) is a global community dedicated to live coding — the practice of writing and modifying algorithms in real time as a performance art.

TOPLAP HK is the Hong Kong chapter, bringing together musicians, visual artists, hackers, and curious minds across the city. We host workshops, performances, open sessions, and online discussions to explore the intersection of code, music, and art.

Whether you've been live coding for years or you've never written a line of code, you are welcome here. We believe that showing your screens — making the process visible — is at the heart of live coding culture.

Tools

Popular Live Coding Environments

TidalCycles

Pattern-based live music coding in Haskell. Beloved for rhythmic complexity and mini-notation.

SuperCollider

Powerful audio synthesis and algorithmic composition environment with a built-in language (sclang).

Sonic Pi

Ruby-based live coding synth designed for education. Great starting point for beginners.

Hydra

Browser-based live visual coding inspired by modular video synthesis. No installation needed.

Strudel

TidalCycles in the browser. JavaScript implementation — run it instantly from any device.

FoxDot

Python live coding environment using SuperCollider as its audio backend.
